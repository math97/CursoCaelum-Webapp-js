CSS-------Cascading stylesheet
    declara estilos
    css busca elemento e formata

    notações:
        chama no html usando tag link: <link href="" rel="stylesheet">
        pode chamar no arquivo css pelos seguintes seletores:
            notação principal: seletor{propriedades:valor}
            diretamente a tag:
                h1{formatações aqui}
            por ID:
                #nomedoide{formatações aqui}
            por CLASS:
                .nomedaclasse{formatações aqui}
            ponto e virgula(;) separa os dados dentro das chaves{};

        estiliza
        animações
